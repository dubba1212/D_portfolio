C) DRAG-TO-CONNECT MODE (MAIN NEW FEATURE)
========================================================
Interaction concept:
User drags from a module chip (center board) to an Output Stream item (right panel) to “connect” them.
When connected:
- Show a permanent thin neon wire line between the module and the output item.
- The output item shows “CONNECTED” status badge.
- The module shows a small check/LED “linked” status.
- Connecting required items unlocks the next category tab (progression).

HOW TO DO DRAGGING (simple and stable):
- On mousedown (or pointerdown) on a module:
  - Start dragging state `dragging = true`, set `sourceModule`
  - Show a temporary “ghost wire” line following the cursor (SVG overlay)
- On pointer move:
  - Update cursor position and ghost line endpoint
- On pointer up:
  - If pointer is over a valid Output Stream drop target:
    - Add a connection record { moduleId, outputId }
    - Play a short “snap” animation (scale pulse)
  - Else:
    - Cancel drag; wire fades out

Drop targets:
- Make each Output Stream item a droppable region with a data-output-id.
- Add hover feedback: output item glows when draggable wire is near it.

MOBILE:
- Drag should still work with touch.
- Also add a fallback: tapping a module then tapping an output creates a connection (for mobile usability).
