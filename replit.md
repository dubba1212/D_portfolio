# ESPortfolio

## Overview
A personal portfolio website for Dubba Srikanth Reddy built with Next.js, featuring animations, particle effects, and a modern dark theme with space/astronaut aesthetics.

## Tech Stack
- **Framework**: Next.js 14 (Pages Router)
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion, React Three Fiber, tsparticles
- **UI Components**: Swiper, React Icons, React Fast Marquee

## Project Structure
- `pages/` - Next.js pages (index, contact, api)
- `components/` - Reusable React components (Avatar, Nav, Layout, etc.)
- `styles/` - Global CSS with Tailwind
- `public/` - Static assets (images, SVGs)
- `variants.js` - Framer Motion animation variants

## Development
- **Dev server**: `npm run dev` (runs on 0.0.0.0:5000)
- **Build**: `npm run build`
- **Start**: `npm run start`

## Deployment
- Target: Autoscale
- Build: `npm run build`
- Run: `npm run start`
